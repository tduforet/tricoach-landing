L'instructeur

À partir d’un modèle que vous choisissez dans notre catalogue, l’instructeur vous indique clairement rang après rang quelles sont les actions à réaliser sur votre pièce. Plus besoin de cahier pour suivre votre avancement.

L'e-learning

L’e-learning vous propose des fiches pratiques pour apprendre les bases du tricot ainsi qu’un module vous permettant d’apprendre de nouveaux points en les pratiquant.

Le compteur de rangs

Envie de suivre un modèle que vous possédez sur un catalogue chez vous ? Le compteur de rang vous permet de suivre sa réalisation en vous indiquant le nombre de rangs que vous avez réalisé, le pourcentage d’avancement global et le nombre de rang avant un prochain évènement.

Les performances

Grâce au suivi de vos performances, vous avez accès à plusieurs indicateurs de progrès: vitesse de tricot, nombre de points acquis, nombre de modèles réalisés… Une motivation de plus pour progresser !
